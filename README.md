# nippon_colors

Implementing all 250 traditional colors for easy use. Inspired by [Nippon Colors](https://nipponcolors.com/)

## Usage
```dart
import 'package:nippon_colors/nippon_colors.dart'

Color ro = NipponColors.nipponColor250;

```
